ExperienceBridgeMod is a plugin for ExperienceMod that adds supports for three popular
Minecraft Mods: RedPower2, IndustrialCraft and EquivalentExchange.

    
TO INSTALL

    Copy the ExperienceBridgeMod.jar file to the Bukkit plugin folder.


PERMISSIONS

 experiencebridgemod.rewards.processing:
   Rewards processing items with processors in IndustrialCraft.

 experiencebridgemod.rewards.transmute:
   Rewards transmuting items with EquivalentExchange.

    
HOW TO COMPILE

    If it's not already referenced, include the Bukkit MOD API:
        bukkit-1.2.5-R4.0.jar
